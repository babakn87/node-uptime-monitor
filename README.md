# Node.js Uptime Monitor

A simple command-line utility to monitor website availability in real-time.

## Features
- Real-time HTTP status checking.
- Modular architecture with separate logging logic.
- Automated file logging with timestamps.
- Interactive CLI: Switch target URL without restarting (using `Ctrl+U`).
- Network error handling.

## Installation

1. Clone the repository:
   ```bash
   git clone [https://github.com/babakn87/node-uptime-monitor.git](https://github.com/babakn87/node-uptime-monitor.git)
2.Install dependencies & run:
  ```bash
   npm install
   node core.js
